API Documentation

  • GET /records (Requires authentication)
    Retrieve all records.

  • POST /records (Requires authentication and permission)
    Create a new record.

  • GET /records/{id} (Requires authentication)
    Retrieve record with given ID.

  • PUT /records/{id} (Requires authentication and permission)
    Replaces record with given ID.

  • DELETE /records/{id} (Requires authentication and permission)
    Deletes record with given ID.
